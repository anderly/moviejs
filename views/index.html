MovieJS

An IMDB-backed movie api written in node.js

Try It Now

About

MovieJS is an experimental movie api built on top of Node.js and Express. It features fast searching of IMDB by title id or keyword with a Redis-backed sliding cache. JSON is currently the only supported response format. Future plans may include actor, showtime info as well as movie info from additional sources besides IMDB.

Resources

  • Titles: /v1/titles/:id (where id is IMDB title id) example: The Dark Knight Rises
  • Titles (search): /v1/titles/?search=:term (where term is a keyword or phrase) example: Search for "katy perry"
